# ESP32烧录工具 V1.00

> 开发环境:python

通过修改乐鑫开发环境中的固件烧录脚本实现脱离开发环境进行烧录esp32固件的功能

**1. 将拨码开关拨到固件烧录状态**  
2. 运行upload_esp32.py脚本  
3. 终端显示脚本运行目录中所有esp32烧录文件  
4. 选择烧录文件  
5. 选择烧录串口  
6. 烧录开始  
7. 烧录结束后断电，将拨码开关拨到app运行态，断电再上电即运行新的esp32程序
